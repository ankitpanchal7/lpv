== Description ==
Fetches the upcoming events from the 3rd party API services using json.

== Installation ==
1. Unpack the entire contents of this plugin zip file into your wp-content/plugins/ folder locally
2. Upload to your site
3. Navigate to wp-admin/plugins.php on your site (your WP plugin page)
4. Activate this plugin



WP CLI Command:
wp lps import_events


Shortcode:
[lps_upcoming_events] User can use this shortcode to render upcoming events.

Upcoming events exported json file.
URL: http://testbg.local/wp-content/plugins/loop-events/upcoming-events.json
The link of this json file will be rendered on the top left corner of the upcoming events shortcode table.

Event automatically exported when wp cli command executed.
